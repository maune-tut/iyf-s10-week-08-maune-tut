## Project Description
A modular React application built using Vite that serves as the foundation for the CommunityHub platform. This project focuses on component-based architecture, passing dynamic data via props, and organizing a professional layout.

## Technologies Used
- React (JSX)
- Vite
- CSS3
- JavaScript (ES6+)
- Git Bash

## Features
- **Component Architecture:** Separated UI into reusable Header, Footer, Sidebar, and PostCard components.
- **Dynamic Data:** Used React Props to display unique content across multiple post cards.
- **Flexbox Layout:** Implemented a left-aligned sidebar with a responsive main feed section.

## How to Run
1. Clone this repository
2. Open your terminal in the project folder
3. Run `npm install` to install dependencies
4. Run `npm run dev` to start the development server
5. Open `http://localhost:5173` in your browser

## Lessons Learned
While building this project, I learned how to move from static HTML/CSS to a component-driven mindset. I mastered the concept of "Props" to pass data from a parent component (App) to child components (PostCard), making the code much more efficient and dry.

## Challenges Faced
I encountered a MIME type error when first running the dev server, which I solved by ensuring the correct script type was set in `index.html` and using the Vite dev server instead of Live Server. I also managed PowerShell execution policy issues by switching to Git Bash for a smoother development experience.
